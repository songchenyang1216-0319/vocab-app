# 四六级地铁背单词 App 使用说明

## 1. 怎么启动

先进入项目目录：

```bash
cd D:\vocab-app
```

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

启动后在浏览器打开终端里显示的地址，通常是：

```text
http://127.0.0.1:5173
```

开发模式适合写代码和本地调试，但 PWA 离线缓存通常要用打包后的预览或正式部署环境测试。

## 2. 怎么打包

执行：

```bash
npm run build
```

打包成功后会生成：

```text
dist/
```

`dist` 目录就是可以部署的静态网站文件。

当前构建会生成 PWA 文件：

```text
dist/manifest.webmanifest
dist/sw.js
dist/workbox-*.js
```

如果看到 bundle 超过 500KB 的提示，暂时不用紧张。当前词表 Markdown 被打进前端包里，所以体积会偏大；后续可以改成构建时生成 JSON 再优化加载。

## 3. 怎么部署

这个项目不需要后端，部署静态文件即可。

常见部署方式：

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages
- 自己的 Nginx 静态服务器

部署时只需要把 `dist` 目录作为网站根目录。

部署前先运行：

```bash
npm run build
```

本地预览生产包：

```bash
npm run preview
```

然后打开预览地址，通常是：

```text
http://127.0.0.1:4173
```

PWA 在正式使用时建议部署到 HTTPS 地址。手机浏览器通常要求 HTTPS 才能完整支持 service worker 和添加到主屏幕。

## 4. 手机怎么使用

手机打开部署后的 HTTPS 地址。

首页可以看到：

- 总词数
- 今日已背
- 累计认识
- 错词数量

主要页面：

- `开始背单词`：每次显示一个英文单词，点击“显示释义”后查看中文。
- `复习错词`：查看错误次数大于 0 的单词。
- `搜索单词`：可以输入英文或中文释义搜索。
- `设置`：修改每日目标、背诵范围、背诵顺序，导入或导出学习记录。

添加到主屏幕：

Android Chrome：

1. 用 Chrome 打开 App 地址。
2. 点击右上角菜单。
3. 选择“添加到主屏幕”或“安装应用”。
4. 桌面会出现“背单词”。

iPhone Safari：

1. 用 Safari 打开 App 地址。
2. 点击底部分享按钮。
3. 选择“添加到主屏幕”。
4. 桌面会出现“背单词”。

离线使用：

1. 第一次必须联网打开 App。
2. 等页面加载完成后，PWA 会缓存核心资源。
3. 下次断网时，可以再次打开已经缓存过的 App。
4. 学习记录保存在本机浏览器的 `localStorage`，不需要登录。

## 5. 如何更新词表

当前 App 使用的词表文件是：

```text
src/data/CET4_CET6_5500_words_CN.md
```

词条格式必须保持类似：

```text
0001. **a** — art. 一（个）；每一（个） `[四级补充]`
0002. **abandon** — vt. 离弃，丢弃；遗弃，抛弃；放弃 n. 放任；纵情 `[四/六]`
```

支持的标签只有：

```text
四/六
六级
四级补充
```

更新步骤：

1. 替换或编辑 `src/data/CET4_CET6_5500_words_CN.md`。
2. 保持每个单词一行，行首使用 4 位编号。
3. 运行构建检查：

```bash
npm run build
```

4. 如果构建通过，再重新部署 `dist` 目录。

如果词表格式错误，解析函数会在浏览器控制台给出清晰提示。可以先打开：

```text
/vocab-preview
```

查看前 20 个解析结果，并确认总数量是否接近 5500。

## 6. 本地数据说明

学习记录保存在：

```text
metro-vocab-progress-v1
```

设置项保存在：

```text
metro-vocab-settings-v1
```

这些数据都在当前浏览器本地。换浏览器、清除浏览器数据或换手机后，本地记录不会自动同步。

建议定期在设置页点击：

```text
导出学习记录 JSON
```

需要恢复时，在设置页点击：

```text
导入学习记录 JSON
```
