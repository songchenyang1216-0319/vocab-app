# 四六级地铁背单词 PWA

一个面向四六级备考的手机端背单词 Web App。项目优先考虑在地铁、排队、课间这类碎片时间里快速使用：打开即学、按钮够大、数据保存在本机浏览器里，也可以作为 PWA 添加到手机主屏幕。

## 技术栈

- React
- TypeScript
- Vite
- React Router
- vite-plugin-pwa
- localStorage

## 本地运行

```bash
npm install
npm run dev
```

启动后按终端提示打开本地地址即可。

## 打包

```bash
npm run build
```

打包产物默认生成在 `dist` 目录。

## 预览打包结果

```bash
npm run preview
```

## 主要功能

- 四六级词表解析和展示
- 首页学习概览
- 今日任务系统
- 背单词流程
- 复习页
- 错词本
- 生词本
- 搜索单词
- 个人笔记
- 学习统计
- 学习数据导入 / 导出
- PWA 离线访问和新版本提示

## 数据存储说明

当前版本不需要后端，也不需要登录。学习记录、今日任务、生词本、个人笔记、设置项等数据都保存在当前浏览器的 `localStorage` 中。

这意味着：

- 换浏览器或换手机时，需要先在设置页导出备份，再导入到新设备。
- 清理浏览器站点数据可能会删除学习进度。
- 词库 Markdown 文件是 App 自带资源，不会被学习数据备份修改。
